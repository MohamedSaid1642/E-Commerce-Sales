# E-Commerce-Sales
## Project Overview

**Data Pipeline Development & ETL Process:** This project demonstrates a complete data engineering workflow implementing a Bronze-Silver-Golden (BSG) architecture to process and analyze online retail transaction data. The pipeline performs comprehensive data cleaning, type optimization, and categorical segmentation, transforming raw sales records into three distinct datasets (Sales, Refunds, and Discounts) enriched with calculated metrics such as profit margins and average pricing. The cleaned data is validated and loaded into a SQL Server database using SQLAlchemy, enabling downstream analytics and reporting on transaction patterns across multiple business dimensions.
